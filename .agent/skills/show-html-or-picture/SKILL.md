把本地 HTML 或图片展示给前端。

规则：

- 优先直接使用 `user/` 目录中的文件。
- 如果文件不在 `user/` 目录下，先复制进去，再引用复制后的路径。
- 只输出最终可渲染的一行内容，不要加解释、代码块或其他文字。
- 路径必须是相对路径，且只能引用 `user/` 目录，不能使用绝对路径或 `/` 开头路径。
- 默认复制，不移动；除非用户明确要求，否则不要输出复制命令。

HTML 输出：

`<iframe src="./user/xxx.html" width="100%" height="400" />`

图片输出：

`![description](./user/xxx.png)`
